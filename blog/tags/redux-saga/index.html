}: {
data: any;

// NB: –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ –Ω–∞ —Å—Ç—Ä–æ–∫—É –∫–æ–¥–∞ `outerCallback({ message, emit });` –±—É–¥–µ—Ç –±–æ–Ω—É—Å
